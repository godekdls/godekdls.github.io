## !!!! 중요 !!!!

Spring Cloud Sleuth의 마지막 마이너 버전은 3.1이다. 최신 커밋은 [3.1.x](https://github.com/spring-cloud/spring-cloud-sleuth/tree/3.1.x) 브랜치에서 확인할 수 있다.

> Spring Cloud Sleuth는 Spring Boot 3.x부터 동작하지 않는다. Spring Cloud Sleuth가 지원하는 마지막 스프링 부트 메이저 버전은 2.x다.

이 프로젝트의 코어 로직은 [Micrometer Tracing](https://micrometer.io/docs/tracing) 프로젝트로 옮겨졌으며, 여러 가지 계측 기능들은 [Micrometer](https://micrometer.io/)와 각 하위 프로젝트들로 이전할 예정이다 (이제는 단일 레포지토리에서 모든 계측 기능들을 개발하지 않는다).

Spring Cloud Sleuth에서 Micrometer Tracing으로 마이그레이션하는 방법은 [Micrometer Tracing 마이그레이션 가이드](https://github.com/micrometer-metrics/tracing/wiki/Spring-Cloud-Sleuth-3.1-Migration-Guide)를 참고하면 된다.
